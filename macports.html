MacPorts. Установка на macOS

Аннотация:  MacPorts. Установка и примеры использования на macOS.
Дата:  Апрель 2022.
Статус статьи:  pg_stat.


MacPorts – система для установки программного обеспечения, распространяемого в исходных кодах, для операционной системы macOS; включает в себя как приложения с командным интерфейсом, так и программы с графическим интерфейсом пользователя (X11 или Aqua). MacPorts распространяется под лицензией BSD и имеет командный интерфейс пользователя.

При помощи этой системы можно установить множество полезных консольных утилит, вроде nmap, mc, nano, wget и т.п. Для установки MacPorts необходимо предварительно установить Xcode и Xcode Command Line Tools, версии которых соответствуют версии системы macOS:

На мой взгляд самым правильным способом установки является установка из пакета, который можно скачать с сайта  https://xcodereleases.com  (необходимо будет ввести Apple ID).

После установки Xcode следует установить Xcode Command Line Tools. Для этого набираем в терминале следующую команду:

$ xcode-select --install

Появится панель с подтверждением установки (нажимаем Install).

А теперь скачиваем и устанавливаем пакет MacPorts (который соответствует версии системы macOS) с сайта  https://www.macports.org.

 

Примеры использования

Вывод списка доступных портов:

$ port list

Поиск необходимой программы:

$ port search nano

Просмотр описания порта:

$ port info nano

Компиляция и установка программы:

$ sudo port install nano

Обновление дерева портов:

$ sudo port -v selfupdate

Обновление установленных портов:

$ sudo port upgrade outdated

Вывод списка портов, которые зависят от nmap:

$ port dependents nmap

Удаление установленного порта:

$ sudo port uninstall nano

Некоторые порты могут зависеть от сервера X11 (он же X Window System – оконная система, обеспечивающая стандартные инструменты и протоколы для построения графического интерфейса пользователя). В этом случае следует установить пакет XQuartz с сайта  https://www.xquartz.org.
